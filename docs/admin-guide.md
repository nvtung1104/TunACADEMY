# Tổng quan cho Quản trị viên

Tài khoản Admin có quyền cao nhất trên hệ thống — quản lý toàn bộ người dùng, lớp học, môn học và năm học.

## Dashboard Admin

| Khu vực | Nội dung |
|---|---|
| **Tổng quan** | Số liệu thống kê toàn hệ thống |
| **Người dùng** | Quản lý tài khoản học sinh và giáo viên |
| **Lớp học** | Tạo và quản lý các lớp học |
| **Năm học** | Quản lý chu kỳ học tập |
| **Môn học** | Danh mục môn học |
| **Bài học** | Xem toàn bộ bài học trong hệ thống |
| **Đề thi** | Xem toàn bộ đề thi trong hệ thống |
| **Bài tập** | Xem toàn bộ bài tập trong hệ thống |
| **Phòng trực tuyến** | Quản lý các phòng học Live |
| **Báo cáo** | Thống kê và xuất dữ liệu |

## Quy trình thiết lập hệ thống mới

```
1. Tạo Năm học
2. Tạo Môn học
3. Tạo Lớp học → gán Năm học + Môn học
4. Tạo tài khoản Giáo viên → phân công lớp chủ nhiệm
5. Tạo tài khoản Học sinh → xếp vào lớp
```

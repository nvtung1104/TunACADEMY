# Tổng quan cho Giáo viên

Sau khi đăng nhập bằng tài khoản giáo viên, bạn được chuyển đến Dashboard tại `/teacher/dashboard`.

## Dashboard giáo viên

| Khu vực | Nội dung |
|---|---|
| **Tổng quan** | Số lớp đang dạy, bài học đã tạo, bài tập chờ chấm |
| **Lớp chủ nhiệm** | Lớp học do bạn phụ trách |
| **Bài học** | Quản lý bài học đã tạo |
| **Bài kiểm tra** | Quản lý đề thi |
| **Bài tập** | Giao và chấm bài tập |
| **Điểm số** | Bảng điểm tổng hợp của học sinh |
| **Phòng trực tuyến** | Tạo và quản lý lớp học Live |

## Quy trình làm việc tiêu biểu

```
Tạo bài học → Gán vào lớp học
→ Giao bài tập / tạo đề thi
→ Chấm bài, nhập điểm, nhận xét
→ Theo dõi tiến độ cả lớp
```

## Menu điều hướng

Sidebar bên trái gồm:

- **Tổng quan** — Dashboard chính
- **Lớp chủ nhiệm** — Quản lý lớp học
- **Bài học** — Tạo và quản lý bài học
- **Bài kiểm tra** — Tạo và quản lý đề thi
- **Bài tập** — Giao và chấm bài tập
- **Điểm số** — Bảng điểm
- **Học trực tuyến** — Phòng học Live
